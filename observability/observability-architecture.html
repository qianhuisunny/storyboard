2. Storage Structure
Edit Events
events/edits.jsonl
Append-only log of every user edit. One JSON object per line. Includes field name, before/after values, diff metrics, and semantic category.
Snapshots
snapshots/{stage}_v{n}.json
Full stage content at key moments: AI generation and human approval. Enables side-by-side diff visualization.
Summary
summary.json
Computed edit patterns per project: fields edited, edit rates, semantic patterns detected. Updated on demand.
